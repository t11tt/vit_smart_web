# Simple Portal Demo

A minimal full-stack demo built with Node.js, Express, and SQLite.

## Features

- Email registration
- User login
- Redirect to the main dashboard after successful login
- Simple software-style UI
- Local SQLite database storage

## Tech Stack

- Node.js
- Express
- SQLite
- HTML
- CSS
- JavaScript

## Project Structure

```bash
simple-portal/

└─ public/
   ├─ index.html
   └─ dashboard.html
